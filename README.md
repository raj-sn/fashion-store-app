# 👗 Fashion Store Mobile Application

A modern and responsive mobile application designed for a fashion store, allowing users to browse products, manage accounts, and interact with a Firebase-powered backend system.

---

## 🚀 Overview
This application was developed as a **real-world freelancing project**, focusing on delivering a clean user interface, efficient navigation, and seamless backend integration using Firebase.

---

## 👨‍💻 My Role
- ✅ Full-Stack Developer  
- ✅ UI/UX Designer  
- ✅ Firebase Integration  
- ✅ Implemented client requirements from design to deployment  

---

## ✨ Key Features

- 🛍 **Product Browsing**
  - View fashion items with images and details

- 🔐 **User Authentication**
  - Secure login & registration using Firebase

- 👤 **User Profile Management**
  - Manage account details easily

- ☁️ **Firebase Integration**
  - Real-time backend database

- 📱 **Responsive UI**
  - Clean, modern, and user-friendly design

---

## 🛠 Technologies Used

- **Framework:** Flutter  
- **Backend:** Firebase  
- **Language:** Dart  
- **Tools:** Android Studio, VS Code  
- **Design:** Figma
